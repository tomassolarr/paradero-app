# paradero-app

Aplicación web desarrollada en React para consultar el estado de paraderos del sistema de transporte público de Santiago, Chile.

## Características

- Búsqueda de paraderos por código
- Información de buses en tiempo real
- Favoritos guardados en localStorage
- Diseño responsive

## Tecnologías

- React 19
- Vite
- React Router
- CSS nativo

## API

Utiliza la API pública de Red Movilidad (xor.cl) para obtener información de paraderos.

## Instalación

```bash
npm install
npm run dev
```
